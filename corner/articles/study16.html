什么是 RLHF

基于人类反馈的强化学习 (RLHF) 在大语言模型的训练中起到了很重要的作用，目的是为了将大模型和人类偏好对齐，使得模型表现得更 “像” 人。RLHF 可以大致分为三个阶段：

  1. 有监督微调 (SFT)：在有监督数据集上对基础大模型进行微调；
  2. 训练奖励模型 (Reward Model)：RLHF 的核心是基于人类反馈训练一个独立的奖励模型，该模型可以估计人类对任何给定的响应的评分；
  3. 用奖励模型优化大模型：大模型基于策略生成对给定 prompt 的反馈，奖励模型给予该反馈输出奖励值，然后用强化学习算法来优化策略。

大模型中的强化学习

强化学习其实是 agent 和环境交互并从环境反馈中学习的过程：agent 在环境的某个状态 (state) 下采取某个动作 (action)，动作改变环境的状态，环境给予奖励 (reward)，然后 agent 根据奖励值来从环境中学习。

而语言模型如何通过强化学习来训练，主要的问题还是在于状态和动作是什么。对于大模型，输入 prompt 后，大模型会输出序列 $y=\{y_t\}_{t=1}^T$，而实际上大模型是逐 token 输出的，即根据 prompt 和当前已输出的 token 来预测下一个 token。所以对于大模型，在时刻 $t$，状态就是 prompt 和 $y_{1:t-1}$ 的结合（$t=1$ 时刻就只有 prompt），动作就是决定下一个 token，即 $y_{t+1}$。
